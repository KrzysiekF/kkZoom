kkScrollAnimate

jQuery plugin to animate website/app elements on scroll

About

kkScrollAnimate to plugin do jQuery, który pozwala na kierowanie animacją elementów strony lub aplikacji za pomocą scrolla.

invert_colors
CSS Class

Możesz dodać / usunąć klasę css do dowolnego elementu na stronie / aplikacji w zależności od pozycji scrolla.

picture_in_picture
Element in viewport

Animacja może być także uruchomiona w momencie, gdy inny element strony pojawi się w viewport.

open_in_browser
Animation on scroll

Kierowanie animacją elementów strony lub aplikacji za pomocą scrolla.

Installation

1. kkScrollAnimate can be installed using bower or you can simply download plugin source. 

$ bower install kkScrollAnimate --save

2. Include JS (ideally at the bottom of the page but also works in the <head>) 

<script src="PATH_TO_FILE/dist/kkScrollAnimate.min.js"></script>

Options

property default description
scrollContent jQuery object $('body') Selektor elementu w którym jest scroll. Najczęście jest to 'window', 'document' lub 'body'
startFromElement jQuery object false Selektor elementu strony, który ma się pojawić w viewport, aby animacja się rozpoczęła.
scrollType string 'vertical' Typ scrollowania elementu lub strony: 'vertical' lub 'horizontal'
startScroll int 0 Wartość scrolla po osiągnięciu której ma się rozpocząć animacja
endScroll int 0 Wartość scrolla przy której animacja ma się zakończyć
cssProperty string '' Właściwość CSS, która ma być manipulowana podczas animacji
before int 0 Początkowa wartość właściwości CSS
after int 0 Końcowa wartość właściwości CSS
unit string '' Jednostka wartości, którą animujemy
